🧱 block.py — класс клетки
Файл содержит класс Block, который представляет одну клетку игрового поля.

Основные обязанности класса:
- Отрисовка клетки
- Обработка кликов мыши
- Отображение текста (число, флаг, бомба)
- Хранение состояния клетки

🔹 Основные методы:
#### onBlock(x, y)
    1. Проверяет, находится ли курсор внутри клетки.
    2. Используется для определения кликов по конкретному блоку.
####  onClick(ind)
    1. Проверяет нажатие кнопки мыши:
        0 — левая кнопка
        2 — правая кнопка
>📌 Используется polling (pg.mouse.get_pressed()),
поэтому в main.py реализована защита от зажатия кнопок.
#### GetPos():
    1. Возвращает позицию клетки в формате: (y, x)

> 📌 Формат выбран осознанно для совместимости с numpy,
где индексация идёт как [row, column].

#### SetColor(blockColor, borderColor)
    1. Меняет цвет клетки и сохраняет предыдущие значения 
        (используется при установке и снятии флага).

#### SetText(text)
    1. Используется для установки текста внутри клетки:
        (число мин рядом, "!", "B").

#### Mark()
    1. Ставит или убирает флаг:
        меняет цвет клетки
        устанавливает символ "!"
    2. при повторном вызове возвращает предыдущее состояние

🧰 func.py — вспомогательные функции
#### Control(x, y, max_X, max_Y)
    1. Проверяет, находятся ли координаты в пределах игрового поля.
    2. Используется:
        при подсчёте бомб
        при рекурсивной очистке пустых клеток

#### Equals(A1, A2)
    1. Проверяет, содержат ли два списка одинаковые элементы
        (используется для проверки победы — все бомбы отмечены).

> ⚠️ Порядок элементов не учитывается.

#### Grey(x)
    1. Возвращает оттенок серого: (x, x, x)

#### Change(num)
    1. Возвращает цвет клетки в зависимости от значения:
        9 — бомба
        0 — пустая клетка
        1–8 — количество мин рядом

🎮 main.py — игровая логика

Основные этапы:
- Создание игрового поля (numpy)
- Случайная генерация бомб
- Подсчёт мин вокруг каждой клетки
- Создание визуальных блоков (Block)
- Обработка кликов:
    - ЛКМ — открыть клетку
    - ПКМ — поставить/убрать флаг
- Проверка победы
- BombCounter(array, x, y)
    - Считает количество бомб вокруг клетки
(в радиусе 1 клетки).

#### CleanPlace(...)
    Рекурсивно открывает соседние клетки, если выбранная клетка пустая (0).
    Глубина рекурсии ограничена (i < 4),
